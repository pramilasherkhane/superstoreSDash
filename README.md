# Superstore Sales Dashboard | Power BI

## Project Overview
This project analyzes Superstore sales data to understand sales performance, profit trends, and customer purchasing behavior. An interactive dashboard was built using Microsoft Power BI to visualize key business metrics and provide actionable insights.

## Key Highlights
- Total Orders: 1.6M
- Total Sales: 22K
- Total Profit: 175K
- Average Ship Days: 4

## Key Insights
- West region contributes the highest share of total sales.
- Consumer segment generates the largest portion of revenue.
- Office Supplies category records the highest sales among product categories.
- Phones, Chairs, and Binders are among the top-performing sub-categories.
- Cash on Delivery is the most commonly used payment method.

## Sales Forecast
A 15-day sales forecast was created using Power BI forecasting techniques to predict future sales trends based on historical data.

## Tools & Technologies Used
- Microsoft Power BI
- Data Analysis
- Data Visualization
